React Js and Node Js application, following along [Brad Traversy tutorial](https://www.youtube.com/watch?v=PBTYxXADG_k)

## Installation

Clone the repo:

```sh
npm run dev
```
The repository uses concurrently Js to run both the React Front-end and NodeJS backend
